# nqm-wrangler-nquirer-mappings

##### databot input
different mappingType "inputSchema"
e.g.

```javascript
mappingType:"serviceIdToLsoa"
```
mappingType is one of "defaultMappingTypes"
defaultMappingTypes are: 
* "serviceIdToLsoa";
* "patients-mapping";
* "service-mapping";
* "5m-neighbour-mapping";
* "ons-mapping";
